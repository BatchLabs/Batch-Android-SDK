# About

Batch is the leading mobile engagement & push notification suite engineered for the mobile era.

# Requirements
 - Android Studio 3.0+
 - Android 15+

# Installation
Gradle (recommended)

```
compile 'com.batch.android:batch-sdk:+'
```

Manual  
You can find the AAR [here](https://batch.com/download/android)
